# タスク: TagList表示コンポーネント実装

## 概要

Todo一覧やTodo詳細でタグを視覚的に表示するコンポーネントを実装する。
タグクリックでフィルタリングを開始できる機能も含む。

## 実装スコープ

- [ ] TagList コンポーネントの作成
- [ ] タグバッジの視覚デザイン（色分け）
- [ ] タグクリック時のフィルタ動作
- [ ] タグ削除機能（編集モード時）
- [ ] 基本的なコンポーネントテストの追加

## 実装スコープ

1. TagList コンポーネントが実装されている
   - Props: `{ tags: Tag[], onTagClick?: (tag: Tag) => void, onTagRemove?: (tagId: string) => void }`
   - タグがバッジとして表示される
2. タグの色分けが実装されている
   - タグ名のハッシュ値から色を自動生成
   - 視覚的に区別しやすい配色
3. タグクリックでフィルタリング
   - onTagClick が指定されている場合、タグクリックでコールバック実行
   - フィルタ状態の管理は親コンポーネント側
4. タグ削除機能（オプション）
   - onTagRemove が指定されている場合、× ボタンを表示
   - × ボタンクリックで削除コールバック実行
5. 基本的なテストが追加されている
   - 正常系: タグ表示、タグクリック、タグ削除

## 技術的考慮事項

- タグ名から色を生成する関数を utils に追加
- ハッシュ関数は簡易的なもの（例: 文字コードの合計 % 色数）
- 配色はアクセシビリティを考慮（背景と文字のコントラスト）
- タグバッジのスタイルは Tailwind CSS で実装
- レスポンシブデザイン対応（タグが多い場合は折り返し）
